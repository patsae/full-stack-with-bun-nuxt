# Pages

ใน Nuxt.js จะใช้ระบบ file-based Routing ในการจัดการระบบ Route (หน้าเว็บ) เพียงแค่สร้างไฟล์หรือโฟลเดอร์ไว้ภายใน pages/ ก็จะถูกเปลี่ยนเป็น Route (URL) โดยอัตโนมัติ เช่น

- app/pages/index.vue 👉 สอดคล้องกับเส้นทาง / (หน้าแรก)
- app/pages/about.vue 👉 สอดคล้องกับเส้นทาง /about
- app/pages/course/index.vue 👉 สอดคล้องกับเส้นทาง /course
- app/pages/course/nuxt-fullstack.vue 👉 สอดคล้องกับเส้นทาง /course/nuxt-fullstack

# 🎯 The Main Entry (app.vue)

- ไฟล์ app/app.vue จะทำหน้าที่เป็นโครงสร้างหลัก (Main Shell) ของทั้งเว็บไซต์
- ต้องใส่คอมโพเนนต์ <NuxtPage /> ไว้ใน app.vue เพื่อบอกให้ Nuxt รู้ว่า "ให้เอาเนื้อหาจากไฟล์ในโฟลเดอร์ pages มาแสดงตรงจุดนี้"
- แต่ถ้าหากคุณลบไฟล์ app.vue ทิ้งไปเลย Nuxt จะใช้ pages/index.vue เป็นหน้าหลักให้โดยอัตโนมัติ

# 🚀 ขั้นตอนการเริ่มใช้งาน (Quick Start)

2. เปิดการแสดงผล: ในไฟล์ app/app.vue ให้เรียกใช้ <NuxtPage />

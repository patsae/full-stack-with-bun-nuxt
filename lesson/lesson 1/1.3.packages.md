# 📦ติดตั้ง Packages

หลังจากติดตั้ง Nuxt.js เสร้จแล้ว ต่อไปเราจะมาติดตั้ง package ที่จำเป็นสำหรับการพัฒนาเว็บแอปพลิเคชั่น

### 📦Typescript

```
bun add -D vue-tsc typescript tsx
```

### 📦Tailwindcss

```
bun add tailwindcss @tailwindcss/vite
```

สร้างไฟล์ main.css ไว้ที่ app/assets/css/main.css

```
mkdir app/assets/css
type nul > app/assets/css/test.css
```

ตั้งค่าเพิ่มเติมใน nuxt.config.ts

```
export default defineNuxtConfig({
  ...,
  css: ["~/assets/css/main.css"],
  vite: {
    plugins: [tailwindcss()],
  },
});
```

### 📦Nuxt Modules อื่นๆ ที่แนะนำ

@nuxt/fonts สำหรับจัดการแสดง font (ใช้ font จาก [google font](https://fonts.google.com/))  
@nuxt/icon สำหรับจัดการแสดง icon ในเว็บ (ใช้ icon จาก [icones.js.org](https://icones.js.org/) )  
@nuxtjs/i18n สำหรับช่วยในการพัฒนา application ให้สามารถรองรับได้หลายภาษา  
@vueuse/core ไลบรารีรวมฟังก์ชันสำเร็จรูป สำหรับ Vue.js 3  

```
bun add @nuxt/fonts @nuxt/icon @nuxtjs/i18n @vueuse/core
```

ตั้งค่าเพิ่มเติมใน nuxt.config.ts

```
export default defineNuxtConfig({
  ...,
  modules: ["@nuxt/icon", "@nuxtjs/i18n", "@nuxt/fonts"],
  icon: {
    mode: "css",
    cssLayer: "base",
  },
  fonts: {
    families: [{ name: "Bai Jamjuree" }],
  },
  i18n: {
    defaultLocale: "th",
    locales: [{ code: "th", name: "Thailand", file: "th.json" }],
  },
});
```

สำหรับ @nuxtjs/i18n ให้สร้างไฟล์ i18n/locales/th.json ไว้ที่ root project

```
mkdir i18n/locales
type nul > i18n/locales/th.json
```

ไฟล์ th.json

```
{
  "welcome": "ยินดีต้อนรับ"
}

```

### 🖼️Nuxt UI

```
bun add @nuxt/ui
```

ตั้งค่าเพิ่มเติมใน nuxt.config.ts

```
export default defineNuxtConfig({
  ...,
  modules: [..., '@nuxt/ui'],
});
```

ตั้งค่าเพิ่มเติมใน app/assets/css/main.css

```
@import "tailwindcss";
@import "@nuxt/ui";
```

ครอบหน้าเพจของเราด้วย <UApp>
เปิดไฟล์ app/app.vue

```
<template>
  <UApp>
    <NuxtPage />
  </UApp>
</template>
```

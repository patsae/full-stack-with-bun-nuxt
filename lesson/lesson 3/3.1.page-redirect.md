# การทำ Page Redirect (เปลี่ยนหน้า) ใน Nuxt 3

จากโครงสร้างและการทำงานในไฟล์ `app/pages/routing/page-one.vue` และ `app/pages/routing/page-two.vue` จะพบว่ามีการใช้เทคนิคการเปลี่ยนหน้า (Redirect/Navigate) หลักๆ อยู่ด้วยกัน 3 วิธี ดังนี้ครับ

## 1. การใช้ `<NuxtLink>`
วิธีนี้เป็นวิธีมาตรฐานสำหรับสร้างลิงก์ใช้ภายในแอป ทำหน้าที่เสมือนแท็ก `<a>` ใน HTML แต่มีความเร็วกว่าเนื่องจากเป็นการเปลี่ยนหน้าแบบ Client-side Routing (ไม่ต้องโหลดรีเฟรชหน้าเว็บใหม่ทั้งหมด)

**โครงสร้างในโค้ด:**
```html
<!-- ในทั้ง page-one และ page-two มีการเขียนเรียกใช้ NuxtLink -->
<NuxtLink to="/routing/page-two" class="btn-block">
  วิธีที่ 1: เปลี่ยนหน้าด้วย NuxtLink
</NuxtLink>
```
**ข้อดี:** เข้าใจง่าย และถูกออกแบบมาสำหรับการทำงานร่วมกับ Router ของ Nuxt อย่างสมบูรณ์แบบ

---

## 2. การใช้ฟังก์ชัน `navigateTo()`
วิธีนี้เป็น Utilities ของ Nuxt 3 โดยเฉพาะ (เป็น Auto-imported function ไม่ต้อง import เอง) เหมาะสำหรับการถูกเรียกใช้แบบ Programmatic Navigation เช่น การเปลี่ยนหน้าเมื่อฟังก์ชันทำงานเสร็จ หรือการคลิกปุ่มที่ต้องการทำงานอย่างอื่นด้วยก่อนการย้ายหน้า

**ใน `page-one.vue`:**
```vue
<template>
  <button @click="toPageTwo()">
    วิธีที่ 2: เปลี่ยนหน้าด้วย Function navigateTo()
  </button>
</template>

<script setup lang="ts">
  const toPageTwo = () => {
      // ใช้ navigateTo เป็น Auto-imported function ของ Nuxt
      return navigateTo('/routing/page-two')
  }
</script>
```
**ทริค:** `navigateTo()` สามารถเรียกใช้ได้ทั้งใน Client และ Server (เช่นการ Redirect ใน Route Middleware)

---

## 3. การใช้ Vue Router (`useRouter().push()`)
เป็นอีกหนึ่งการเปลี่ยนหน้าในลักษณะ Programmatic Navigation โดยดึง instance ของ Vue Router ตัวหลักมาใช้งาน (รูปแบบดั้งเดิมจาก Vue 3) 

**ใน `page-two.vue`:**
```vue
<template>
  <button @click="toPageOne()">
    วิธีที่ 2: เปลี่ยนหน้าด้วย Function router.push()
  </button>
</template>

<script setup lang="ts">
  // ใช้ useRouter คู่กับ .push()
  const router = useRouter()

  const toPageOne = () => {
      router.push('/routing/page-one')
  }
</script>
```

### สรุปความแตกต่างของทั้ง 3 วิธี:
- **`NuxtLink`:** ใช้ประกอบตรงๆ ในหน้า View (Template HTML) เหมาะสุดสำหรับการทำลิงก์ปกติ
- **`navigateTo()`:** เป็นท่าเฉพาะเจาะจงของ Nuxt ใช้ใน `<script>` หรือ Middleware เขียนได้ง่ายและรองรับ Server-side ด้วย
- **`useRouter().push()`:** เป็นท่าของ Vue Router มาตรฐาน ใช้เมื่อต้องการการจัดการเส้นทางขั้นสูงหรือเฉพาะเจาะจงตาม Vue Router API
